# Pet Clínica Veterinária do Dr. Thiago Mello

Site estático de demonstração comercial criado para apresentar uma página de conversão profissional, responsiva e focada em contatos para a Pet Clínica Veterinária do Dr. Thiago Mello.

## Como abrir

Abra o arquivo `index.html` diretamente no navegador.

Também é possível servir a pasta com qualquer servidor estático local, mas isso não é obrigatório.

## Formulário

O formulário é estático e não armazena dados. Ao enviar, ele valida os campos obrigatórios e abre o WhatsApp com uma mensagem pronta contendo:

- Nome do tutor.
- Nome do pet.
- Telefone / WhatsApp.
- Tipo de atendimento desejado.
- Mensagem opcional.

## Depoimentos

A seção de depoimentos usa exemplos fictícios apenas para apresentação comercial do site. Eles estão identificados como ilustrativos e devem ser substituídos por avaliações reais autorizadas antes da publicação.

## Referências e cuidados de conteúdo

O PDF `Atendimento clínico básico.pdf` foi usado para estruturar serviços comuns de uma clínica veterinária, como consulta clínica, vacinação, check-up, cuidados ambulatoriais, acompanhamento cirúrgico e avaliação oncológica.

O site `https://veros.vet/` foi acessado em 21/06/2026 e usado apenas como referência de hierarquia, ritmo visual, presença de chamada para ação e organização de serviços. Nenhum texto ou identidade visual foi copiado.

## Paleta visual

A paleta foi inferida do logo e dos materiais fornecidos:

- Azul vivo para proximidade e identidade visual.
- Azul-marinho para confiança e autoridade clínica.
- Amarelo para chamadas de ação.
- Verde para saúde, cuidado e sinalização positiva.
- Branco quente e cinza claro para leveza e leitura.
